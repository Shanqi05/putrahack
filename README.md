## ⚡ TLDR (Super Quick)

```bash
# Terminal 1 - Backend
cd server && npm install && npm start

# Terminal 2 - Frontend  
cd client && npm install && npm run dev
```

**Then:**
1. Create `server/.env` with your Google AI key
2. Open http://localhost:3001
3. Done! 🎉
